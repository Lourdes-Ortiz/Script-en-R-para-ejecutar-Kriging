Para que el script creado en R realice la interpolación y posterior creación del raster, 
es necesario que la base de datos cumpla de manera estrica con las siguientes columnas y sus respectivos títulos, 
tomando en cuenta formato (mayúsculas y guiones):  
- PRECIO_POR_M2
- AÑO_DE_TRANSACCION
- LAT_DD
- LON_DD
